# Actividad Sumativa 1 Llanquihue Tour Digitalizacion - Desarrollo Orientado A Objeto I

## Autor Del Proyecto
- **Nombre:** Hector Eduardo Soto Vasquez
- **Carrera:** Analista Programador Computacional

## Descripcion Del Proyecto
Este Proyecto Se creo debido al problema que habia de una empresa llamada lanquihue tour en el cual esta misma hacia todo manualmente por escrito causando conflictos de duplicar informacion Y una desorganizacion real ya que no habia forma de organizarlo cada uno


## Estructura General Del Proyecto

```plaintext
📁 src/
├── app/           #Clase Principal Con El Metodo Main
├── model/         #Clase De Dominio (Personas,Cliente,Empleado,Operario,Provedor,etc.)
├──Excepciones/    #Clase Para la carga de excepciones
````

---

##Instrucciones para ejecutar el proyecto

1. Clonar el repositorio desde Github.

```bash
git clone https://github.com/HectorSotoDuocuc/LlanquihueTourDigitalizacion-TrabajoSumativo1.git
```

2. abre el proyecto en IntelliJ IDEA.

3. Ejecutar el archivo `Main.java` desde el paquete `app` buscando el boton verde de play que estara arriba a la derecha.
